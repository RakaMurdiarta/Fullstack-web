Mengenal Pengertian Website, Ketahui Jenis dan Fungsinya

Merdeka.com - Website adalah kmpulan halaman web yang dapat diakses publik dan saling terkait yang berbagi satu sama domain. Website dapat dibuat dan dikelola oleh individu, grup, bisnis, atau organisasi ntuk melayani berbagai tujuan. Bersama-sama, semua website yang dapat diakses publik membentuk World Wide Web. Meskipun terjasang disebut "halaman web", definisi ini salah, karena website terdiri dari beberapa halaman web. Website memiliki variasi yang hampir tidak ada habisnya, termasuk situs pendidikan, situs media sosial, situs e-commerce, dan sebagainya. Halaman-halaman dalam sebuah website biasanya merupakan campuran teks dan media lainnya. Meskipun demikian, tidak ada aturan yang mengatur bentuk website. Untuk melihat halaman websit membutuhkan sebuah browser. Contoh browser di antaranya adalah sebagai berikut:

  1. Internet Explorer
  2. Edge
  3. Safari
  4. Chrome
